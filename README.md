# Sidekiq::Profiling

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sidekiq-profiling'
```

## Usage

Simply having the gem in your Gemfile is enough to get you started.

## Configuring

Depends on Sidekiq >= 2.9.0
